# React native work hours chart

Small React Native chart examples built with Expo + TypeScript.

## Why this repo exists

This repo is a tiny showcase of reusable mobile chart components.
Developers can copy a chart component, pass data, and use it in their own app.

## Included examples

- Weekly work hours bar chart
- Duration formatting helpers
- Simple summary card

## Tech

- Expo
- React Native
- TypeScript
- Victory Native XL
- Reanimated
- Skia

## Install

```bash
npm install
npx expo start
```

## Libraries Used

- `victory-native` — chart components
- `react-native-reanimated` — animation support
- `react-native-gesture-handler` — gesture handling
- `@shopify/react-native-skia` — chart rendering
- `npx expo install react-native-worklets` - needed in newer Expo/Reanimated setups
