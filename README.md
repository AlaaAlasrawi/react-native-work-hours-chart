# React native work hours chart

Small React Native chart examples built with Expo + TypeScript.

## Why this repo exists

This repo is a tiny showcase of reusable mobile chart components.
Developers can copy a chart component, pass data, and use it in their own app.

## Included examples

- Weekly work hours bar chart
- Duration formatting helpers
- Simple summary card

## Tech

- Expo
- React Native
- TypeScript
- Victory Native XL
- Reanimated
- Skia

## Install

```bash
npm install
npx expo start
